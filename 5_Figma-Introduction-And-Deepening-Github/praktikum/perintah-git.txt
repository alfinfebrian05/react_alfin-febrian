1. Git init : menginisialisasi folder untuk di tambahkan ke repository
2. Git add <nama-file> : menambahkan file / folder ke dalam sebuah directory repository
3. Git commit -m "isi-pesan-commit" : membuat commit untuk mengupload isi folder ke repository
4. Git remote add origin <url> : menambahkan alamat url dari git repository yang telah kita buat
5. Git merge <nama-file> : menggabungkan file yang telah kita buat di branch lainnya
6. Git branch <nama-branch> : menambahkan branch baru
7. Git branch -D <nama-branch> : menghapus branch
8. Git checkout branch <nama-branch> : pindah dari branch satu ke branch lainnya
9. Git diff : melihat perbedaan versi dan perubahan apa saja yang telah dilakukan
10. Git stash : memperbarui isi file tanpa perlu push ke repository
11. Git stash apply : memperbarui isi file tanpa perlu push ke local git
